# 🪔 Trikala Darshana — ತ್ರಿಕಾಲ ದರ್ಶನ

**AI/ML Startup Survival Intelligence Platform**

> *Trained on Failure. Built for Survival.*

## About
Trikala Darshana predicts whether an Indian startup will **Survive or Fail** using Machine Learning trained on 60,432 real Indian startup records.

## Model Performance
- ✅ Accuracy: 87.2%
- ✅ F1 Score: 0.92
- ✅ ROC AUC: 0.90

## Tech Stack
Python · FastAPI · Streamlit · Random Forest · XGBoost · SMOTE · SQLite · pandas · scikit-learn

## Run Locally
```bash
uvicorn main:app --reload --port 8000
streamlit run dashboard.py
```
